Boise DateFort

Fun little hackathon project built in a couple of days during Hackfort 2015. Datefort uses the Boise Parks open data set to randomly select an activity and the Yelp API to randomly select a nearby dinner option to suggest date ideas for people in Boise.

I first created a simple backend with Spring Boot (the team preferred Java) then as we simplified the app, it became Javascript-only with no backend. I took on a DevOps role towards the end, building a web server in the cloud, pushing updates up to it as we debugged and fixed code in the last minute.

It turns out people really liked the app - and we won the Hackfort 2015 hackathon!

Papa Punk Blog

This is another personal project. Papa Punk is a little space on the web to share stories from my punk rock past and for others to chime in with their own related stories and thoughts.
